
This project is a React + TypeScript + Vite demo featuring a floating Danmaku (bullet chat) component and a Message Card modal.

## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```

2. **Start the development server:**
   ```sh
   npm run dev
   ```

3. **Open in browser:**
   Visit [http://localhost:5173](http://localhost:5173) to view the app.

## Build for Production

```sh
npm run build
```

## Features
- Floating pill-shaped Danmaku with glassmorphism effect
- Click to pause and show message details in a modal
- Like button with animation, image lightbox
